README CONTENT:Form Validation Project
 Project Overview:

This project is a simple Registration Form built using HTML, CSS, and JavaScript.
It validates user input before submitting the form to make sure all details are correct.

 Features:
User-friendly registration form
Input validation using JavaScript
Alerts for incorrect data
Clean and simple UI design
 Technologies Used:
HTML – Structure of the form
CSS – Styling and layout
JavaScript – Validation logic
 Form Fields:
Name
Email
Password
Phone Number
 Validation Rules:
Name should not be empty
Email must be in valid format (example@gmail.com
)
Password must contain at least 6 characters
Phone number must be exactly 10 digits
How It Works:
User fills the form
Clicks on Submit button
validateForm() function is triggered
Each input field is checked:
If invalid → error alert is shown
If all valid → success message is displayed
Project Structure:
project-folder/
index.html   (Main file with HTML, CSS, JS)
How to Run:
Download or copy the code
Save it as index.html
Open the file in any web browser
Fill the form and test validation
 Advantages:
Prevents wrong data entry
Easy to understand and use
Beginner-friendly project
Improves frontend skills
 Future Improvements:
Add error messages below input fields instead of alerts
Use advanced validation (strong password, OTP, etc.)
Connect with backend (database storage)
Add responsive design for mobile
Conclusion:

This project helps beginners understand how form validation works in web development.
It is a basic but important step in building real-world applications.
